# Bookstore-System-Analysis-and-Design

## Overview

This project presents the analysis and design of a Bookstore Management System for both in-store operations and e-commerce. The project follows the Software Development Life Cycle (SDLC), from business requirement analysis and system modeling to database and interface design.

The proposed solution aims to improve bookstore operations by supporting sales management, inventory control, customer management, order processing, financial reporting, and online shopping while addressing limitations identified in existing solutions such as SAPO and MISA eShop.

---

## Objectives

- Analyze existing bookstore management processes.
- Identify business requirements and system limitations.
- Design a scalable information system supporting both physical and online bookstores.
- Model business processes and system interactions using UML.
- Design a relational database and user interface prototypes.
